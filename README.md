# OSC-OSDI-Lab
This repository contains lab implementations for the Operating System Capstone (Operating System Design and Implementation) on the Raspberry Pi 3B+ platform using bare-metal programming.

The firmware images in the `firmware` folder can be tested directly on a Raspberry Pi 3B+ or emulated using QEMU. For labs after Lab 2, the bootloader image must be loaded first, and the kernel image is then transferred via UART.

lab1 - Hello World  
lab2 - Booting Program  
lab3 - Exception and Interrupt  
lab4 - Allocator
